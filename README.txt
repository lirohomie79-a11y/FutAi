FutAI - App de Previsões de Futebol

Tema: Preto + Verde
SDK mínimo: 21 (Android 5.0)
Compatível com AIDE

Funcionalidades:
- Jogos de Hoje e Amanhã
- Todos os mercados de apostas:
    - Vitória / Empate / Derrota
    - Over / Under
    - Ambas Marcam
    - Casa / Fora
    - 1º tempo / 2º tempo
- Percentuais variando de 1% a 100%
- Dados reais via API-Football (chave necessária)
